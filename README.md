# DeepDeep

## Configure ENV VAR
- LOCO_uuid : Device UUID
- LOCO_device : Device Name
- LOCO_email : Kakaotalk Email
- LOCO_pwd : Kakaotalk Password

## Install Dependencies
- install node.js
```
npm install
```

## Start
```
npm start
```
this will compile `src/index.ts` file to `dist` and run `dist/index.js`
